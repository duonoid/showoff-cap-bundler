This is a presentation using ShowOff (http://github.com/schacon/showoff)

To see it, install showoff and then run it in this directory:

  $ gem install showoff
  $ git clone git://github.com/spovich/showoff-cap-bundler.git
  $ cd showoff-cap-bundler
  $ showoff serve

Then you should be able to go to http://localhost:9090 to view the presentation.
