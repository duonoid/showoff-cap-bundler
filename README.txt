This is a ShowOff presentation (http://github.com/spovich/showoff-cap-bundler)

If you want to see it, you need to install showoff and then run it in this directory:

  $ gem install showoff
  $ git clone git://github.com/spovich/showoff-cap-bundler.git
  $ cd showoff-cap-bundler
  $ showoff

Then you should be able to go to http://localhost:9090 to view the presentation.
